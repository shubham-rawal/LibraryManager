## LibraryManager

A C++ program that implements Data File Handling and uses this concept to create a fully functional LibraryManagement system. This features a beautiful and interactive Menu-Driven program, in which users may input information regarding new books, authors, publishors, etc.
These entries may be fetched by using one of the above-mentioned fields.

>Note: This is just a demo-project and is still being worked upon, although the program is functional, it is not completely efficient.
